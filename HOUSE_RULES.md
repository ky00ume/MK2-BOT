# 🃏 UNO 하우스 룰 가이드

> **미쿠나이스트라이 UNO** — 플레이어 vs 하츠네 미쿠 (AI)

---

## 📋 기본 규칙

표준 UNO 규칙을 기반으로 합니다.

- **카드 내기 조건**: 현재 색상 또는 현재 숫자/기호가 일치하는 카드를 낼 수 있습니다.
- **와일드 카드**: 항상 낼 수 있으며, 다음 색상을 직접 선택합니다.
- **+4 와일드**: 항상 낼 수 있으며, 색상 선택 후 상대방이 카드 4장을 뽑습니다.
- **+2 카드**: 상대방이 카드 2장을 뽑고 턴을 건너뜁니다.
- **스킵 카드**: 상대방의 턴을 건너뜁니다.
- **리버스 카드**: 2인 전용 규칙 적용 (아래 하우스 룰 참고).

---

## 🏠 하우스 룰 (이 게임만의 특수 규칙)

### 1. 2인 전용
이 게임은 **플레이어 1명 vs 미쿠(AI) 1명**의 맞대결 전용입니다.

### 2. 3판 2선승제
먼저 **2판을 이기는 쪽**이 매치 승리입니다. 최대 3판까지 진행됩니다.

### 3. 리버스 = 스킵
2인 게임이므로 **리버스 카드는 스킵 카드와 동일**하게 처리됩니다. 상대방 턴이 건너뛰어집니다.

### 4. UNO 선언 시스템
- 패가 **2장**일 때 화면에 **UNO 버튼**이 표시됩니다.
- 카드를 내어 패가 **1장**이 되기 **전에** UNO 버튼을 눌러야 합니다.
- 미선언 시, 상대방이 **😈 UNO 안 외쳤어!** 버튼을 눌러 **+2 패널티**를 줄 수 있습니다.

### 5. 미쿠의 UNO 선언은 자동
미쿠(AI)는 패가 1장이 되면 자동으로 UNO를 선언합니다. 패널티 버튼은 미쿠가 UNO를 선언하지 않은 경우에만 활성화됩니다.

### 6. 1% 특수 이벤트 🌟
- 미쿠가 **UNO 선언을 3회 이상** 달성하면 **1% 확률**로 특수 이벤트가 발동됩니다.
- 이벤트 발동 조건: 플레이어가 **그린 8** 카드를 내는 순간
- 이벤트 내용: 플레이어의 패가 모두 드로우 카드로 교체되어 미쿠에게 대량의 드로우가 쏟아집니다.
- 미쿠는 패닉 상태로 무너지는 RP를 수행합니다.

### 7. 벌칙 RP
- 매치가 끝난 후 **패배한 쪽에 대한 벌칙 RP**가 진행됩니다.
- 벌칙 RP 완료 후 하단의 **다시 하기** 버튼을 눌러 새 매치를 시작합니다.

### 8. 덱 소진 시 자동 리셔플
뽑기 덱이 소진되면 버려진 카드 더미를 자동으로 섞어 덱을 새로 만듭니다.

## 🎮 조작법

| 조작 | 방법 |
|------|------|
| **게임 시작** | 채팅창에 `/start` 입력 |
| **카드 내기** | 황금 빛(glow)이 활성화된 카드를 클릭 |
| **카드 뽑기** | **카드 뽑기** 버튼 클릭 |
| **UNO 선언** | 패가 2장일 때 표시되는 **UNO!** 버튼 클릭 |
| **색상 선택** | 와일드 카드 사용 후 빨강/노랑/초록/파랑 버튼 클릭 |
| **다음 판 시작** | 라운드 종료 후 하단 **다음 판 시작** 버튼 클릭 |
| **재도전** | 매치 종료 후 벌칙 RP 진행 뒤 하단 **다시 하기** 버튼 클릭 |

> 💡 황금 빛(glow)이 없는 카드는 현재 낼 수 없는 카드입니다.

---

## 🏆 승리 조건

- **라운드 승리**: 먼저 패를 모두 내는 쪽이 해당 라운드 승리
- **매치 승리**: 3판 중 **2판을 먼저 이기는 쪽**이 최종 승리
- 매치 승리 시 승자/패자에 따른 벌칙 RP 진행
