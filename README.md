### 카드 소개

이 카드는 **ぬぬぬぬぬぬぬぬ feat. 初音ミク — "みむかｩわナイストライ(Mimukauwa Nice Try)"** 뮤직비디오 컨셉을 기반으로 한 RISUAI 인터랙티브 RP 카드입니다.

뮤비에서 하츠네 미쿠는 **메스가키(めすガキ)** 속성으로, 애인(= {{user}})과 UNO를 하며 끊임없이 도발합니다. 패가 딱 1장뿐인 압도적인 우위 속에 기고만장하지만, 상대방이 그린8 + 드로우 카드 13장 콤보로 역전에 성공하고, 미쿠는 현실을 부정하며 판을 뒤엎고 도망칩니다.

**카드 특징:**
- 💬 메스가키 하츠네 미쿠와의 RP (역안, 오버사이즈 흰 셔츠, 고양이 오버니삭스)
- 🎮 **LUA Trigger Script 기반** 완전한 UNO 게임 엔진 (하우스 룰 포함)
- 🖱️ **클릭만으로 게임 진행** — 텍스트 입력 없이 카드 버튼 클릭으로 플레이
- 🏆 3판 2승제 시리즈
- 🎲 1% 확률 뮤비 재현 이벤트
- 🖥️ 동적 HTML/CSS UNO 게임 UI (listenEdit 기반 실시간 렌더링)
- 🔞 성인 NSFW 허용 ("뭐든 들어줄게"의 결과 포함 가능)

### 🏗️ 아키텍처

| 구성 요소 | 역할 |
|-----------|------|
| **LUA Trigger Script** | 게임 엔진 전담: 덱 관리, 턴 로직, 상태 저장, UI 렌더링, 버튼 클릭 처리 |
| **AI (LLM)** | 미쿠 캐릭터 RP 서사만 담당. 게임 상태 출력 금지 |
| **Regex Script** | AI 출력에서 깨진 태그(`{AFF|...}`, `{SIG|...}`, `{null}`) 자동 제거 |

> ℹ️ **이전 버전과의 차이**: 이전 버전은 AI가 `[UNO_STATE]` 블록을 텍스트로 출력하면 Regex Script(내부 JavaScript)가 파싱하는 방식이었으나, **RISUAI에서 `<script>` 태그가 실행되지 않아 동작 불가** 상태였습니다. 현재 버전은 모든 게임 로직을 LUA 트리거 스크립트로 이전하여 이 문제를 해결했습니다.

### 📦 설치 방법

#### 1단계: 캐릭터 카드 임포트

1. `MikuNiceTry_CharCard.json` 파일을 다운로드합니다.
2. RISUAI를 열고 **"Import Character"** 기능을 사용하여 JSON 파일을 임포트합니다.
3. 카드가 정상적으로 불러와지면 `初音ミク | みむかｩわナイストライ` 카드가 캐릭터 목록에 나타납니다.

> 💡 LUA Trigger Script와 Regex Script가 카드 JSON 내부에 내장되어 있습니다. 임포트 즉시 자동 적용됩니다.

#### 2단계: RISUAI 설정 확인

1. RISUAI → **Settings** → **Trigger Script** 기능이 활성화되어 있는지 확인합니다.
2. HTML 렌더링이 허용된 상태인지 확인합니다 (기본값: 허용).

### 🎮 사용법

1. RISUAI에서 `初音ミク | みむかｩわナイストライ` 카드로 새 채팅을 시작합니다.
2. 미쿠의 첫 인사(도발)를 읽고, **`/start`** 를 입력하여 UNO 게임을 시작합니다.
3. 게임 UI가 렌더링되면 **카드 버튼을 클릭**하여 카드를 냅니다.

**클릭 기반 조작:**
- 손패에서 **낼 수 있는 카드** → 버튼으로 강조 표시됨 → 클릭으로 제출
- **낼 수 없는 카드** → 흐릿하게 표시 (비활성)
- **🂠 뽑기 버튼** → 덱에서 카드 뽑기
- **🎴 UNO! 버튼** → 패가 1장 남을 때 나타남 → UNO 선언

**텍스트 입력도 지원 (선택):**
- `빨강 5` / `Red 5` — 빨강 5 카드 내기
- `초록 스킵` / `Green Skip` — 초록 스킵 내기
- `와일드` / `Wild` — 와일드 카드 내기
- `드로우` / `draw` — 덱에서 1장 뽑기
- `UNO!` — UNO 선언
