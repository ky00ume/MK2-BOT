### 카드 소개

이 카드는 **ぬぬぬぬぬぬぬぬ feat. 初音ミク — "みむかｩわナイストライ(Mimukauwa Nice Try)"** 뮤직비디오 컨셉을 기반으로 한 RISUAI 인터랙티브 RP 카드입니다.

뮤비에서 하츠네 미쿠는 **메스가키(めすガキ)** 속성으로, 애인(= {{user}})과 UNO를 하며 끊임없이 도발합니다. 패가 딱 1장뿐인 압도적인 우위 속에 기고만장하지만, 상대방이 그린8 + 드로우 카드 13장 콤보로 역전에 성공하고, 미쿠는 현실을 부정하며 판을 뒤엎고 도망칩니다.

**카드 특징:**
- 💬 메스가키 하츠네 미쿠와의 RP (역안, 오버사이즈 흰 셔츠, 고양이 오버니삭스)
- 🎮 실제 작동하는 UNO 게임 시스템 (하우스 룰 포함)
- 🏆 3판 2승제
- 🎲 1% 확률 뮤비 재현 이벤트
- 🖥️ HTML/CSS 기반 UNO 게임 UI (Regex Script 연동)
- 🔞 성인 NSFW 허용 ("뭐든 들어줄게"의 결과 포함 가능)

### 📦 설치 방법

#### 1단계: 캐릭터 카드 임포트

1. `MikuNiceTry_CharCard.json` 파일을 다운로드합니다.
2. RISUAI를 열고 **"Import Character"** 기능을 사용하여 JSON 파일을 임포트합니다.
3. 카드가 정상적으로 불러와지면 `초音ミク | みむかｩわナイストライ` 카드가 캐릭터 목록에 나타납니다.

> 💡 카드 JSON 내부에 Regex Script가 이미 포함되어 있습니다. 자동 적용되면 별도 설정 없이 바로 사용 가능합니다.

#### 2단계: Regex Script 수동 설정 (자동 적용이 안 된 경우)

1. RISUAI → **Settings** → **Regex Scripts** → **Add Regex Script**
2. 아래 값을 입력합니다:

| 항목 | 값 |
|------|-----|
| Script Name | `UNO Game UI` |
| Type | `Modify Display` |
| Find (Pattern) | `\[UNO_STATE\][\s\S]*?\[\/UNO_STATE\]` |
| Replace (Output) | `regex_script_uno_ui.md` 파일의 HTML 코드 참조 |

3. **Save**를 누르고 스크립트를 활성화합니다.

자세한 설정 가이드는 [`regex_script_uno_ui.md`](./regex_script_uno_ui.md)를 참조하세요.

### 🎮 사용법

1. RISUAI에서 `初音ミク | みむかｩわナイストライ` 카드로 새 채팅을 시작합니다.
2. 미쿠의 첫 인사(도발)를 읽고, **`/start`** 를 입력하여 UNO 게임을 시작합니다.
3. 게임이 시작되면 UI가 렌더링되고 7장씩 패가 배분됩니다.

**카드 내는 법:**
- `빨강 5` / `Red 5` — 빨강 5 카드 내기
- `초록 스킵` / `Green Skip` — 초록 스킵 내기
- `와일드` / `Wild` — 와일드 카드 내기
- `드로우` / `카드 뽑기` — 덱에서 1장 뽑기
- `UNO!` — UNO 선언

### 🃏 하우스 룰

이 카드에는 뮤직비디오 가사에 명시된 하우스 룰이 적용됩니다:

| 규칙 | 내용 |
|------|------|
| **기호 카드 마감 불가** | 마지막 카드(승리 카드)는 반드시 숫자 카드(0~9)여야 합니다. Skip, Reverse, Draw Two, Wild, Wild Draw Four로는 게임 종료 불가. |
| **드로우 카드 스태킹** | Draw Two와 Wild Draw Four를 연속으로 쌓아서 사용 가능합니다. 누적된 드로우 장수가 다음 플레이어에게 전달됩니다. |
| **UNO 선언** | 패가 1장 남았을 때 "UNO!"를 외쳐야 합니다. 외치지 않고 다음 턴이 진행되면 벌칙 2장. |

### 🏆 3판 2승제

- 먼저 **2라운드**를 이기는 쪽이 최종 승자입니다.
- 최대 3라운드가 진행됩니다.
- 각 라운드 종료 후 스코어보드가 업데이트됩니다.
- **미쿠가 이기면**: 극도로 기고만장, {{user}}에게 패배 인정을 요구합니다.
- **{{user}}가 이기면**: 미쿠 대붕괴 → "뭐든 들어줄게" 이행 → 이후 RP는 {{user}} 자유입니다.
